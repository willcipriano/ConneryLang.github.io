# Strings

### upper
StdLib
{: .label .label-green }
upper converts any lower case characters to upper case.
```
upper "this is lower case"
```
```
"THIS IS LOWER CASE"
```

### lower
StdLib
{: .label .label-green }
lower converts any upper case characters to lower case.
```
lower "THIS IS UPPER CASE"
```
```
"this is lower case"
```
